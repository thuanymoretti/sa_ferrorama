## Sistema Ferroma
O sistema é uma aplicação web desenvolvida com o objetivo de gerenciar usuarios e simular um sistema relacionado ao contexto ferroviario. A plataforma permite o cadastro e autenticação de usuários, servindo como base para futuras funcionalidades do sistema, 

## como executar o projeto
1. baixe ou clone o repositório
2. utilize um servidor local 
3. inicie apache e mysql
4. acesse:
https://github.com/kamigdr/Programa-o-de-Aplicativo.git
